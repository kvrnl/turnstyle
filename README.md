# Turnstyle

**A desktop DJ turntable app — by [KVRNL](https://kvrnl.io).**

Two virtual decks, a crossfader, 3-band EQ, pitch/tempo control, scratching, waveforms, and
beat-sync — a full DJ rig that runs natively on your machine. Load tracks, beatmatch, and perform.

### Features
- Dual decks with spinning platters, jog/scratch, and cue points
- Per-channel 3-band EQ + volume, equal-power crossfader
- Pitch/tempo fader (±8%) with matching platter speed
- Waveform display with click-to-seek and live playhead
- BPM estimate per track + one-click SYNC to beatmatch
- Drag-and-drop loading (mp3, wav, ogg, m4a, aac, flac, webm)
- In-app **Report a Problem** and built-in **auto-updates**
